# LibVF.IO

LibVF.IO is a vendor neutral GPU multiplexing tool driven by YAML & VFIO.

# Features

1. Virtualization of Intel, AMD, & Nvidia GPUs.
2. Simple declarative YAML API.
3. Automated installation.
